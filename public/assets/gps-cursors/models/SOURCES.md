3D GPS cursor models

The navigation cursor uses the GLB models placed in `source/`.

Current runtime bindings:

- `auralith-nav-black`: `source/bmw_m3_coupe_e30_1986.glb`
- `auralith-nav-red`: `source/lamborghini_aventador_lp700.glb`
- `auralith-nav-white`: `source/xyz_school_coursework_highpoly_porsche_singer.glb`
- `auralith-nav-cyan`: `source/nissan_gt-r_2008.glb`
- `auralith-nav-graphite`: `source/nissan_fairlady_z_s30240z_1978.glb`

`gps-car.glb` is kept only as the previous emergency fallback asset and is no longer referenced by the GPS cursor selector.

The source GLB files were provided locally for this project. Keep the original license/source metadata with any replacement models before deploying them publicly.
